fiveone jQuery综合插件
点此下载：jquery.fiveone.min.js
介绍：
作者：zhuCheer
此插件综合了常用的各类页面效果，目前此版本支持图片滚动、页面浮标、文字展开收起效果，效果还在不断更新，敬请关注。
使用前准备：
页面必需调用jQuery和fiveone插件文件，才能正常使用fiveone插件。
开始： 
<script type="text/javascript" src="http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.7.2.min.js"></script>
<script type="text/javascript" src="jquery.fiveone.min.js"></script>//这里请注意文件路径
<script type="text/javascript">
$(function(){

//这里添加你自己的代码

});
</script>
 
示例：
设置一个浮动层在页面上：Demo 
<script type="text/javascript">
$(function(){
$("#floatDiv").floatBox();
});
</script>

使用方法及参数设置：
 
各种效果方法介绍： 
<script type="text/javascript">
$(function(){
$("#floatDiv").floatBox();
});
</script>
效果1：页面浮标效果 $("selected").floatBox(); Demo
注意：使用此浮标时，你的div尽量不要有padding值，否则会出现浮层飘到可见区域以外的情况。
参数名 默认值 说明
speed 20 每次移动单位距离（step）的时间间隔（毫秒），越小效果越流畅。
position {left:50,top:50} 浮层的起始坐标，相对屏幕最左上角。
step 1 每次飘动的最小位移量（像素），越小飘动效果越细腻。
onstop 1 鼠标经过浮层是否停下来，1：停下来 0：不停 

<script type="text/javascript">
$(function(){
$("#rollDiv1").rollAction();
});
</script>
效果2：页面内容横向/纵向滚动效果 $("selected").rollAction() Demo
参数名 默认值 说明
speed 40 每移动一个像素的时间间隔，单位：毫秒，越小速度越快
direction 'left' 滚动方向，有：'up'，'down'，’right‘，’left‘四个选项
onstop 1 鼠标经过浮层是否停下来，1：停下来 0：不停 


<script type="text/javascript">
$(function(){
$("#a").focusImg1();
});
</script>
效果3：图片切换效果 $("selected").focusImg1() Demo
参数名 默认值 说明
speed 3000 图片停留的时间，单位：毫秒
zindex 999 设置对象的z-index值
delay 800 切换时动画效果变化的时间，单位毫秒
cycle 1 是否循环，1：循环，0：不循环（放完所有图片就停下来）
tag "img" 设置产生切换效果的标签，默认是图片切换，为图片加上链接，则此项需改成“a” 


<script type="text/javascript">
$(function(){
$("img").hoverBtn();
});
</script>
效果4：按钮鼠标经过切换图片效果 $("selected").hoverBtn() Demo
说明：选择器（”selected“）选择的元素必须是img标签，必须为img元素添加hover属性，属性值填写鼠标经过的图片地址
参数名 默认值 说明
type “show” “show”：直接切换没有任何效果 “fade”：切换时有半透明效果
